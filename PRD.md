# PRD - Web Viewer de Apresentacao

## Problema
Apresentar resultados do otimizador em Pygame dificulta demonstracao e acesso remoto.

## Objetivo
Fornecer um painel web para demonstracao com leitura rapida de resultados e chat contextual.

## Requisitos Funcionais (atuais)
- Mostrar resumo da ultima simulacao.
- Mostrar KPIs chave.
- Exibir mapa HTML mais recente.
- Exibir relatorio markdown mais recente.
- Permitir chat com contexto da simulacao via Ollama.

## Requisitos Nao Funcionais
- Interface responsiva.
- Consistencia visual com design system.
- Preservar funcionalidade original sem alterar regra de negocio.
- Feedback claro para estados de erro e vazio.

## Fora de escopo
- Alterar algoritmo genetico.
- Alterar modelo de dados.
- Adicionar backend/API nova.
